## 📝 Формати даних

### Поля продукту:
- `id` - унікальний ідентифікатор
- `name` - назва продукту (українська/італійська)
- `type` - тип продукту (фарба, ґрунтовка, лак, емаль)
- `category` - категорія продукту
- `description` - опис (українськими)
- `effect` - описання ефекту покриття
- `base` - основа (водна дисперсія, силоксан, вапно, тощо)
- `finish` - типи фінішу (матовий, глянцевий, атласний, металізований, тощо)
- `colors` - масив об'єктів з кодом кольору та URL текстури
- `colors_count` - загальна кількість доступних кольорів
- `photos` - масив URL-ів фото продукту
- `eco` - позначка екологічності
- `url` - посилання на сторінку продукту на oikos.ua
- `tags` - теги для пошуку
